# Sketch Tech Solutions

A modern, full-stack company website for **Sketch Tech Solutions**, offering professional tech services and an integrated marketplace where users can browse, order, and purchase tech products and request custom-built solutions.

🌐 Live Site: https://sketchtechsolutions.vercel.app

## 🚀 Overview

Sketch Tech Solutions is a digital platform that combines a company website with a fully functional marketplace. Users can explore services, place custom orders, and purchase available tech products directly through the platform.

The system is designed to streamline how clients interact with tech service providers by making ordering, communication, and service delivery seamless and efficient.

## ✨ Features

### 🏢 Company Website
- Professional landing page showcasing services
- About section highlighting company expertise
- Contact section with inquiry forms
- Fully responsive modern UI/UX design

### 🛒 Marketplace
- Browse available tech products
- Product listings with categories
- Product detail pages
- Order / purchase flow (depending on configuration)

### 🧾 Custom Order System
- Submit tailored tech service requests
- Structured order forms for client requirements
- Easy capture of client specifications
- Ready for admin-side processing

### ⚙️ Services Offered
- Software development
- Web application development
- Mobile app development
- IT consulting
- System design & deployment
- Custom tech solutions on demand
