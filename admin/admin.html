⚡ How to Connect Google Sheets
Step 1. Go to sheets.google.com and create a new spreadsheet. Name it SketchTech Products.
Step 2. Click Extensions → Apps Script. Delete everything in the editor.
Step 3. Open the file google_apps_script.js that came with your site files. Copy all of it and paste into Apps Script.
Step 4. Click Run → setupSheet. Allow permissions when asked. This creates your product sheet with all current products.
Step 5. Click Deploy → New deployment → Web App. Set Execute as: Me and Who has access: Anyone. Click Deploy and copy the URL.
Step 6. Paste that URL in the box on this page and click Connect.
Admin password: SKETCH_ADMIN_2026 — change this in the Apps Script before deploying.
